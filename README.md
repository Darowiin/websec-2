# Прогноз погоды — Россия 🌦️

Одностраничный сайт с интерактивной картой, на которой можно выбрать населённый пункт и получить прогноз погоды на 7 дней.

## Возможности

- 🗺️ **Интерактивная карта** с 1000 крупнейших населённых пунктов России
- 📊 **Графики прогноза** температуры, осадков и скорости ветра (Chart.js)
- 🔍 **Поиск** населённого пункта по названию
- 📍 **Кластеризация** маркеров для удобной навигации
- 🌡️ **Текущая погода** с основными показателями
- 📱 **Адаптивный дизайн** для мобильных устройств

## Технологии

- **Leaflet.js** — интерактивная карта с тёмной темой (CartoDB Dark Matter)
- **Chart.js** — графики прогноза погоды
- **Open-Meteo API** — бесплатный API прогноза погоды (без ключа)
- **Leaflet.markercluster** — кластеризация маркеров
- Чистый **HTML + CSS + JavaScript**, без серверного компонента

## Источники данных

- **Населённые пункты**: [tochno.st/datasets/allsettlements](https://tochno.st/datasets/allsettlements) — топ-1000 по населению
- **Прогноз погоды**: [Open-Meteo API](https://open-meteo.com/) — бесплатно, без API-ключа

## Запуск

Для работы необходим HTTP-сервер:

```bash
# Python
python -m http.server 8080

# Node.js
npx -y http-server -p 8080 -c-1
```

Затем откройте [http://localhost:8080](http://localhost:8080) в браузере.
